TürkAnimu İndirici - Kılavuz

Bu scriptin veya kodun veya programın, her ne derseniz, başlıca özellikleri:

  • Yığın ve tekil anime indirme otomasyonu
  • Anime izleme ve izlerken kaydetme
  • Uygulama içinden arama yapmak ve fansub seçebilmek

Manuel Kurulum

Windows
Mpv ve Geckodriver'ın son sürümünü indirip program dizinine atın ya da bulundukları dizini sistem PATH'ına ekleyin. Ardından gerekli paketleri kurun: py -m pip install -r requirements.txt yada pip3 install -r requiremenets.txt. Youtube-dl'i buradan'i indirebilirsiniz, yalnız youtube-dl'in çalışması için VC Redist 2010 kurmanız da gerekli, buradan indirebilirsiniz.

MacOS
Brew paket yöneticisini yükledikten sonra brew install geckodriver mpv komutuyla gerekli uygulamaları kurun. Ardından pip3 install -r requirements.txt ile gerekli paketleri kurun.

Linux
Paket yöneticinizle veya github sayfalarından mpv ve geckodriver'ı kurduktan sonra gerekli paketleri pip3 install -r requirements.txt ile kurun.

İzlerken kaydetme modu

Mpv’nin deneysel bir modülüdür. Ayarlardan aktifleştirip devre dışı bırakabilirsiniz.

Problemler

Programı ilk açışınızda gecikmeler yaşayabilirsiniz. Ara sıra arama yaptığınızda zaman aşımı hatası alabilirsiniz, bu sitenin o sırada yavaş veya bakımda olmasından kaynaklanır. Bunlar dışında programa ilişkin yaşadığınız probleme en hızlı dönüşü github’da issue ticket’ı açarsanız sağlayabilirim.
